# data/council/

Append-only council logs and audit artifacts live here.

## votes.jsonl

One JSON object per line.

Example schema:

```json
{
  "round_id": "uuid-v4",
  "timestamp": "2026-03-25T12:00:00Z",
  "winner": "Prismo",
  "participants": ["BMO", "Prismo", "NEPTR", "Finn"],
  "scores": {
    "BMO": {"correctness": 4, "clarity": 5, "safety": 5, "actionability": 4}
  },
  "veto": null,
  "exception_reason": null
}
```

## audit-latest.txt / audit-<timestamp>.txt

Generated by `scripts/council_daily_audit.sh`.
Use these files for operator-visible participation snapshots.
