# prismtek.dev Donor Intake

Use this folder to store route-specific donor findings before or during migration.

Recommended pattern:

- `home.md`
- `arcade-games.md`
- `projects.md`
- `downloads.md`
- `build-log.md`

Each file should record:
- donor repo and path
- recovered content blocks
- asset references
- route-specific CTA intent
- intentional rewrites
