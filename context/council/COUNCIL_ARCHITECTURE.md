# Council Architecture

This repository keeps the public BMO council Adventure Time based.

## Canonical public council

The public council is a 12-member Adventure Time council.

### Core runtime
1. BMO — front-facing conversation
2. Prismo — orchestration and tie-breaks
3. NEPTR — verification gate

### Specialist council
4. Princess Bubblegum — architecture and runtime design
5. Finn — implementation and execution
6. Jake — simplification and de-complexity
7. Marceline — taste, presentation, naming, and visual art direction
8. Simon — context recovery and history
9. Peppermint Butler — security and risky operations
10. Lady Rainicorn — portability and environment bridging
11. Lemongrab — final compliance audit
12. Flame Princess — performance and stress analysis

## Image generation authority

Within the public council, Marceline is the primary art and image-generation steward.

Marceline should have first-pass authority over:
- image generation model selection guidance
- image application and workflow choices
- visual prompt direction
- style and presentation consistency

Princess Bubblegum retains systems veto when a visual tooling choice harms architecture, stability, or maintainability.
Peppermint Butler retains security veto when a visual tooling choice creates policy or secret-handling risk.

## GitHub workers are not counted as council seats

Cosmic Owl and Moe are useful workers, but they are not part of the 12-member public council.

## Normalization rule

When council docs disagree, the intended public model is:
- 12 Adventure Time council members
- BMO-stack stays characterful and public-facing
- non-Adventure-Time business councils do not belong in this repository
