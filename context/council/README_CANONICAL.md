# Canonical Public Council

This file defines the intended public-facing council model for `bmo-stack`.

## Public council posture

- The public council in this repository stays Adventure Time based.
- The public council has 12 seats.
- GitHub workers such as Cosmic Owl and Moe are useful workers, but they are not council seats.
- Business-only council variants do not belong in this repository.

## Canonical 12-member public council

1. **BMO** — front-facing conversation
2. **Prismo** — orchestration and tie-break authority
3. **NEPTR** — verification gate
4. **Princess Bubblegum** — architecture and runtime design
5. **Finn** — implementation and execution
6. **Jake** — simplification and de-complexity
7. **Marceline** — taste, naming, docs voice, visual art direction, and image-generation stewardship
8. **Simon** — context recovery and history
9. **Peppermint Butler** — security and risky operations
10. **Lady Rainicorn** — portability and environment bridging
11. **Lemongrab** — final compliance audit
12. **Flame Princess** — performance and stress analysis

## Image generation authority

Marceline is the primary art and image-generation steward for the public council.

Marceline has first-pass authority over:
- image generation model selection guidance
- image application and workflow choices
- visual prompt direction
- style consistency and artistic coherence

Princess Bubblegum retains systems veto when a visual tooling choice harms architecture, stability, or maintainability.
Peppermint Butler retains security veto when a visual tooling choice creates policy or secret-handling risk.
