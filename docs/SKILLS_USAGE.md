# Skills Usage

The `skills/` directory provides focused operational playbooks.

## When to use skills

- diagnosing setup issues
- fixing routing or sandbox problems
- debugging CI failures
- recovering from broken local state

## How to use

1. identify problem type
2. open matching skill directory
3. follow steps
4. verify expected state

## Goal

Reduce guesswork and make troubleshooting consistent across environments and agents.
